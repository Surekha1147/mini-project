# 📚 Digital Library Portal – HTML Mini Project

## 📌 Project Overview
This mini project is a structured HTML webpage designed as a Digital Library Portal.  
It demonstrates the use of headings, text formatting tags, div sections, and commonly used HTML attributes.

The project focuses on strengthening core HTML fundamentals.

---

## 🎯 Objectives
- Practice proper heading hierarchy (h1 to h6)
- Use text formatting tags correctly
- Understand div-based layout structure
- Apply HTML attributes effectively
- Explore the `contenteditable` attribute

---

## 🏷 HTML Tags Used

### 🔹 Headings
- `<h1>` to `<h6>` for structured content hierarchy

### 🔹 Formatting Tags
- `<b>` – Bold text  
- `<i>` – Italic text  
- `<u>` – Underlined text  
- `<mark>` – Highlighted text  
- `<small>` – Smaller text  
- `<marquee>` – Scrolling text  

### 🔹 Structural Tag
- `<div>` – Used to divide sections like header, content, and footer

---

## ⚙ Attributes Used
- `id` – Unique identification for sections
- `class` – Grouping elements
- `style` – Inline CSS styling
- `title` – Tooltip information
- `align` – Content alignment
- `contenteditable` – Makes text editable directly in the browser
- `behavior` & `direction` – Used in marquee animation

---

## 💡 Key Concept Learned

### 1️⃣ Heading Hierarchy
Understanding the importance of proper structure from `<h1>` to `<h6>`.

### 2️⃣ Formatting Control
Applying visual formatting using HTML tags.

### 3️⃣ Contenteditable Attribute
Allows users to modify text directly on the webpage without refreshing.

### 4️⃣ Layout Structure
Using `<div>` to organize content into meaningful sections.
